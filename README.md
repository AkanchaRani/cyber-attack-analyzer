## 🚀 Overview  

CyberIDS is a web-based intrusion detection system that analyzes logs to detect suspicious activities such as unauthorized access attempts, malicious patterns, and abnormal behavior.  

It provides a centralized dashboard for monitoring threats and enables real-time actions like blocking malicious IPs.

---

## ✨ Highlights  

- Converts raw logs into meaningful insights  
- Detects real-world attack patterns  
- Clean and intuitive monitoring dashboard  
- Secure authentication system  
- Practical cybersecurity implementation

## ⚡ Features  

| Feature | Description |
|--------|------------|
| Authentication | Secure login and registration |
| Log Upload | Upload system logs for analysis |
| Manual Entry | Add logs manually for testing |
| Detection Engine | Identifies suspicious patterns |
| Dashboard | Visual monitoring of threats |
| IP Blocking | Block malicious IPs |
| Admin Panel | Manage users and system |
