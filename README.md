🚀 Overview

CyberIDS is a web-based intrusion detection system that analyzes logs to detect suspicious activities such as unauthorized access attempts, malicious patterns, and abnormal behavior.

It provides a centralized dashboard to monitor threats and take real-time security actions like blocking malicious IPs.

✨ Highlights
🔍 Converts raw logs → meaningful insights
🚨 Detects real-world attack patterns
📊 Clean and intuitive security dashboard
🔐 Built with secure authentication system
🛠️ Focused on practical cybersecurity implementatio

⚡ Features
Feature	Description
🔐 Authentication	Secure login & registration
